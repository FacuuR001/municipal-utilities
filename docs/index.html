Estructura

Qué incluye este paquete

La navegación queda reducida a lo útil: Inicio, Componentes, Utilidades, Tokens y Ejemplos. Se eliminó la página técnica que no aportaba a la documentación.

Más aire visual

Se aumentaron separaciones entre secciones, cards y bloques para que no se vea todo apretado.

Ejemplos más reales

Se reemplazaron ejemplos flojos por pantallas de gestión, trámites, tablas y formularios.

La página de utilidades se genera desde el CSS y muestra todas las clases detectadas.

Sin filtros raros

En utilidades queda una búsqueda por texto simple; se sacaron los botones de filtro.
